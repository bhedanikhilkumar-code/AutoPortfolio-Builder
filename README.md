# AutoPortfolio Builder

AutoPortfolio Builder converts your GitHub profile + repositories into a polished personal portfolio website.

## Why this project
Developers spend hours writing portfolio content manually. This tool automates:
- project summaries
- tech stack extraction
- featured project selection
- one-click portfolio generation

## Core Features (MVP)
- Connect GitHub username
- Fetch repositories and metadata
- Auto-generate portfolio sections:
  - Hero
  - About
  - Projects
  - Skills
  - Contact
- Export static site (HTML/CSS/JS)

## Future Scope
- Multiple themes
- Custom domain support
- AI rewrite mode for project descriptions
- PDF resume generation from same data

## Suggested Tech Stack
- Frontend: Next.js / React
- Backend: FastAPI
- Data source: GitHub REST API
